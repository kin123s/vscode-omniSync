### WelcomePanel UX 흐름

```
사이드바 클릭 → 중앙 Webview 패널
  ├── Step 1: 플랫폼 카드 선택
  │     🌐 Jira Cloud  |  🖥️ Jira Server
  │     🐙 GitHub      |  📐 Linear (준비 중)
  └── Step 2: 로그인 폼
        Jira Server → Domain + Email + API Token
        Jira Cloud  → OAuth 버튼
        GitHub      → Domain + PAT
```
