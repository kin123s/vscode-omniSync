# 🌐 OmniSync (vscode-omniSync)

멀티 플랫폼 대응 **AI 프로젝트 어시스턴트(Universal Agent)**로 동작하는 VS Code 전용 확장(Extension)입니다. 본 프로젝트는 `prj-vscode-extension` 모노레포의 퍼블릭 서브모듈로 관리됩니다.

## 💡 주요 기능 (Features)

- **다중 트래커 연동**: Jira(Cloud/Server), GitHub Issues, Linear 등 주요 이슈 트래킹 플랫폼을 통합 뷰로 관리.
- **AI-Driven 컨텍스트 분석**: 백그라운드에서 현재 작업 중인 코드의 범위를 추적하고, AI를 통해 지능적으로 코드리뷰 및 진행 상황을 리포팅.
- **현대적인 Webview Dashboard**: 화려하고 인터랙티브한 React + Vite 기반의 프리미엄 UI 탑재.

## 🚀 시작하기 (Getting Started)

1. `pnpm install`을 통해 의존성 설치
2. `pnpm run compile` 또는 `pnpm run watch`로 소스코드 빌드 (esbuild)
3. VS Code `Run and Debug`를 통해 **Run Extension** 실행 (`F5`)

*(추가적인 설정과 상세 보안 지침은 상위 레포지토리의 `.ai/rules.md`를 참고 부탁드립니다.)*
